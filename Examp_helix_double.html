To create an dislocation helix

Step 1:

Use Tools/Fcc to creat an Bcc crystal sample: bcc_reg.lmp, use Tool/Helical_surface to create two mesh files of helical surfaces: mesh0.dat and mesh1.dat

Step 2:

Open terminal, run the following command:

1CryDisGen para_file bcc_reg.lmp

para_file

Content of the Parameter file:
